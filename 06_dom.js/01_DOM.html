window :- {Document}
Document :- {html}
HTML :- {head, body}
head :- {title,meta}
title :-{text node}
meta :-{attribute}
body :- {div}
div :- {attribute, h1, p}
h1 :- {attribute, text node}
p :- {text node}
In console you can make changes in website using document.
Like example
1. document.getElementById('title').getAttribute('class') // console output will be title.
2. document.getElementById('title').setAttribute('class','set') // class attribute will be set to "set" and it will overwrite previous

similarly like this we can make changes in console.
3. title.style.padding="100px"
4. title.style.borderRadius ="100px".
5. title.innerHTML // gives code inside
6. title.innerText // give text which is visible
7. title.innercontent // give content including unvisible text but not html.

// More
document.querySelector("h2") // will give first h2 as output
document.querySelectorAll("h2") // will give nodelist as output witch is like array of all h2. but its not an array some of properties match with array

When you do document.getelementbyclassName('list-item') type of code in console you will get HTMLcollection of all elements with this class but this htmlcollection is also not an array. So to apply array methods on this nodelist and htmlcollections we have to first convert them into an array . by storing them in some variable and then using :=> Array.from(variable). now you can use array methods on them like foreach and map and etc. Dom learning
